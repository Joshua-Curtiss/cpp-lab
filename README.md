# Computational Clinical Psychology Lab Website

Welcome to the GitHub repository for the Computational Clinical Psychology (CPP) Lab website!

## Overview

This is a professional, responsive website for showcasing research, team members, publications, and opportunities related to the CPP Lab.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Multiple Sections**: 
  - Home (Hero)
  - Research
  - People/Team
  - Publications
  - Instruments/Tools
  - News & Updates
  - Join the Lab
  - Participate in Research
- **Smooth Scrolling**: Navigation with smooth scroll animations
- **Mobile-Friendly Navigation**: Hamburger menu for smaller screens
- **Accessibility**: Semantic HTML and accessible color contrasts
- **Professional Styling**: Clean, modern design with institutional color scheme
